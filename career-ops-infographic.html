The Principle
The system finds jobs. You only decide. Finding is automated (Apify + GitHub Actions). Evaluating is AI (career-ops + Claude). Applying is you — but only to A/B grades. 3–5 targeted applications beat 50 generic ones every time. Your personal data (CV, salary targets, phone) never leaves your machine — everything sensitive is gitignored by design.

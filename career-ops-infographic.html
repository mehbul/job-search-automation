🕷️
LinkedIn Scraper
apify-fetch.mjs
Hits the Apify API to scrape LinkedIn for target roles by location and title. Filters by keyword allowlist/blocklist, deduplicates against all previously seen jobs, and writes fresh URLs into a pipeline inbox file.
⏰
Daily Cloud Automation
daily-fetch.yml
GitHub Actions workflow that runs the scraper every morning at 8 AM IST on GitHub's servers — no local machine required. Results are automatically committed back to the repo so my friend just runs git pull.
⚙️
3 Files to Make It Yours
5-min setup
Edit 3 files and it works for any role anywhere: config/profile.yml (your role + location + salary), portals.yml (title filters), cv.md (your CV). That’s it.

🚀 What it does for you — works for any role, any industry, any country
📊
A–F Job Scoring
7-block evaluation: CV match, comp, culture, red flags, posting legitimacy. Scores 1–5. Tells you not to apply below 4.0.
📄
Tailored CV PDFs
JD keywords injected into existing experience. ATS-optimized PDF via Playwright. Never invents anything — only reformulates.
🏢
Company Research
Full intel before applying: funding, layoff history, culture signals, tech stack, Glassdoor reputation. Ghost-job detection built in.
💬
LinkedIn Outreach
Finds the actual hiring manager (not just HR). Drafts a personalised DM based on their profile. Quality-first, not spammy.
🎤
Interview Prep
Company-specific questions + STAR+R story bank that grows with every evaluation. Covers culture fit, red flag answers, tech stack.
📈
Tracks Everything
Full pipeline: Evaluated → Applied → Interview → Offer. Follow-up cadence, rejection patterns, application history. Nothing slips.

I built this for a friend who was spending 2+ hours a day manually scrolling LinkedIn. I thought: this is an engineering problem. It has an engineering solution.

Built the scraping layer using Apify + GitHub Actions. Plugged it into career-ops (open source AI evaluation engine by @santifer) as the intelligence layer. Now LinkedIn is scraped every morning, every role scored A–F by Claude AI, and tailored CVs are ready before they even open their laptop.

Fork it. Change 3 files. It works for your job search too — any role, any country, any language.
